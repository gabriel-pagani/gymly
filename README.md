# Configuration
```
git clone https://github.com/gabriel-pagani/dashly.git && cd dashly/
```
```
make build-system
```
```
make start-system
```
```
make container-terminal python manage.py createsuperuser
```

# How to Use
After configuration, access http://127.0.0.1:8000
